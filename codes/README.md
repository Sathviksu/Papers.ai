# Papers.ai

Papers.ai is a modern web application designed for research paper management, parsing, and comparison. Built with Next.js, Firebase, and AI-driven workflows, it allows users to upload, organize, and extract insights from academic documents efficiently.

## Features

- **Paper Dashboard**: Overview of your research activity and recently added papers.
- **Document Parsing**: Robust extraction of text from PDF, DOCX, and other formats.
- **Library Management**: Search and organize your collection of research papers.
- **Paper Comparison**: Side-by-side comparison of multiple papers to identify key differences and similarities.
- **Export**: Export summaries and insights in various formats.

## Technical Skills & Stack

- **Frontend**: Next.js, React, Tailwind CSS, Framer Motion
- **Backend**: Next.js API Routes (Node.js runtime)
- **Database & Auth**: Firebase (Firestore, Authentication)
- **AI**: Genkit, OpenAI, Groq
- **Testing**: **Jest**, React Testing Library
- **Utilities**: Lucide React, Radix UI

## Recent Highlights

- **Added unit test coverage for core API routes**: Implemented comprehensive Jest testing for the document parsing API, ensuring reliable handling of multiple file formats and robust error reporting.
- **Mobile Responsive Design**: Optimized the UI for seamless experience across all device sizes.
- **Firebase Auth Integration**: Secure authentication flow with persistent session management.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Run tests:
   ```bash
   npm test
   ```
